## 对比选择
对比几种方式，可以发现，速度最快的是方式六手动调用win32 api获取图片数据并使用全局变量传递数据，但是adb可以配合模拟器后台运行，而win32 api要求软件不能后台运行且不能最小化，所以要根据实际需求进行选取
